# Age Calculator

A simple and responsive Age Calculator built using HTML, Tailwind CSS, and JavaScript. It calculates a user's exact age in years, months, and days based on their birth date.

### Technologies Used

* HTML
* Tailwind CSS
* JavaScript

### Features

* Date picker input
* Accurate age calculation
* Responsive design
* Easy-to-use interface
