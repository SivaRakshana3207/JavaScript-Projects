# JavaScript Projects

A collection of mini projects built using HTML, CSS, Tailwind CSS, and JavaScript to strengthen my front-end development skills.

## Project 1: Tip Calculator

A simple and responsive Tip Calculator that calculates the total bill amount based on the entered bill amount and tip percentage.

### Features

* Enter bill amount
* Enter tip percentage
* Automatic tip calculation
* Displays total amount
* Responsive UI
* Modern design using Tailwind CSS

### Technologies Used

* HTML5
* Tailwind CSS
* JavaScript
